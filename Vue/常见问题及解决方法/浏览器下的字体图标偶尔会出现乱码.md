
##### 1. 浏览器下，字体图标，偶尔会出现乱码

解决方法：
```javascript
// vue.config.js
module.exports = {
  css: {
    loaderOptions: {
      sass: {
        sassOptions: {
          outputStyle: 'expanded'
        }
      }
    }
  }
}
```

最终打包后，会在css第一行插入```@charset "UTF-8"```

`参考： https://github.com/PanJiaChen/vue-element-admin/issues/3526`
